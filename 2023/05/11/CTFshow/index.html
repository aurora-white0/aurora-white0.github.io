图片类型题目解题思路

  • winHex 打开查看

    • 文件头尾的缺失和错误

    • 直接搜索 flag，key 等

  • bpg 文件使用对应查看器 honeyview

  • 图片 flag 使用识字工具查看

  • 查看是否图片内含有其他文件

    • kali 中binwalk打开分离

      • binwalk filename查看 binwalk -e filename分离

    • binwalk 无法正确分离，使用foremost

      • foremost filename -o outputfilename

    • 文件自动分离出错时，使用dd 手动分离

      • dd if=filename of=outfilename bs=1 skip=开始分离的字节数 count=分割个数
        • bs：同时设置读写块的大小为 bytes
        • skip：从输入文件开头跳过多少个 blocks 个块后开始复制
        • count：读写块个数个数
      • binwalk 显示 zip 文件从 22895 开始，到 23046 为其结束
        • dd if=sim.jpg of=11.zip bs=1 count=23046 skip=22895
